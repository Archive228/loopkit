# evidence/

Drop a real before/after pair here per skill. `before.md` is a verbatim transcript (prompt + agent response) showing what happens on a representative task WITHOUT this skill loaded — the failure mode the skill exists to prevent. `after.md` is the same prompt with the skill loaded, showing the changed behaviour. No synthetic examples, no cherry-picked evals: one real task, two transcripts. If you cannot produce the pair, the skill is not ready to ship — land it as a draft, gather the evidence, then merge.
